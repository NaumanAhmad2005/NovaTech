## рҹҡҖ Quick Start


## рҹҢҹ Features

### Visual & Design
- вҡ« Premium dark theme with `#050816` deep space background

- рҹ’Ў Neon blue/cyan glow effects on hover

### Animations & Interactions
- рҹҺ¬ **Loading screen** вҖ” rotating hexagonal logo + progress bar
- рҹ“Ҹ **Scroll progress bar** at top of viewport
- рҹҢҠ **Lenis smooth scroll** вҖ” buttery fluid scrolling
- вңЁ **Framer Motion** вҖ” scroll-triggered stagger animations on every section
- рҹғҸ **Card hover** вҖ” lift + scale + border glow + radial gradient
- рҹҺ  **Infinite marquee** вҖ” tech stack scrolling banner

### 3D & Three.js
- рҹҢҗ **Hero Globe** вҖ” wireframe sphere with lat/lon dots, glow rings, mouse-reactive rotation
- вӯҗ **Star field** вҖ” deep space particle background in hero
- рҹ§  **Neural network** вҖ” animated node graph in AI section
- вңЁ **Particle cloud** вҖ” slowly rotating particles

### Backend Integrations
- рҹ“І **WhatsApp Alerts** вҖ” Contact form submissions are instantly delivered to your WhatsApp DM via [CallMeBot API](https://www.callmebot.com/) (free, no account required)
- рҹ—„пёҸ **Newsletter Database** вҖ” Subscriber emails are stored in a [Supabase](https://supabase.com) PostgreSQL table with duplicate prevention (`upsert`)
- рҹ”Ғ **API Routes** вҖ” Two dedicated Next.js API routes handle all backend logic server-side

### рҹ”’ Security
- рҹӣЎпёҸ **XSS Prevention** вҖ” all user inputs stripped of HTML tags, JS protocols, and event handlers before processing
- рҹҡҰ **Rate Limiting** вҖ” contact form: 3 requests per IP per 10 min; newsletter: 5 per IP per hour
- вң… **Server-side Validation** вҖ” all fields re-validated on the server regardless of frontend state
- рҹ”җ **HTTP Security Headers** вҖ” `Content-Security-Policy`, `X-Frame-Options`, `X-Content-Type-Options`, `Referrer-Policy`, `Permissions-Policy`, `X-XSS-Protection`
- рҹ“Ӣ **Whitelist Validation** вҖ” dropdown values (service, budget) validated against an allowed-values list

### Form Validation (Contact Page)
| Field | Rule |
|-------|------|
| Full Name | Letters, spaces, hyphens, apostrophes вҖ” 2 to 60 characters |
| Email | Valid RFC-5322 format вҖ” max 254 characters |
| Phone | Digits, `+`, `-`, spaces, parentheses вҖ” 7 to 15 digits (ITU E.164) вҖ” optional |
| Company | Letters, digits, common symbols вҖ” max 100 characters вҖ” optional |
| Message | Min 10 characters, max 5000 characters with live counter |
| Dropdowns | Whitelist-checked against allowed values on both client and server |
